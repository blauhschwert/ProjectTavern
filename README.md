# 🍺 Idle Tavern Tycoon  
*A cozy idle brewing game made for Godot Wild Jam #92*

## 🎮 About the Game
**Idle Tavern Tycoon** is a relaxing idle/management game where you run your own tavern, brew beer, and grow your reputation.

Starting with a small setup, you’ll:
- Brew different types of beer 🍻
- Serve customers automatically
- Upgrade your tavern and equipment
- Unlock new recipes and profits over time

Even when you're away, your tavern keeps working!

## 🧠 Theme & Jam
Created for **Godot Wild Jam #92**  
Theme interpretation: *Growth / Automation / Time* (adapt if needed)

## ⚙️ Features
- 🕰️ Idle progression system
- 🍺 Beer brewing mechanics
- 💰 Upgrade system for production and profit
- 🧍 Passive customer flow
- 📈 Scaling economy
- 🎨 Simple and cozy visuals

## 🛠️ Built With
- [Godot Engine](https://godotengine.org/) (version 4.x recommended)
